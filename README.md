# Kiki-release

## 简介
Kiki 是一个基于 [Dify](https://dify.ai) 的跨平台桌面客户端，使用 [Tauri](https://tauri.app/) + Vue3 开发。它提供了便捷的 AI 对话体验，支持划词翻译、自定义工具栏等功能。

## 更新日志
### v0.2.1
- 新增划词功能
- 支持自定义工具栏
